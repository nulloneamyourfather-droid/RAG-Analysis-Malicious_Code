# Documentation Layout

`attck_knowledge/docs/` 是当前项目的正式文档目录。

## Structure

- `design/`
  - 系统设计、数据分层和技术路线文档
- `thesis/`
  - 论文章节草稿、替换稿、答辩提纲、致谢
- `maintenance/`
  - 项目清理记录、状态检查、工作区审计

## Notes

- `attck_knowledge/` 根目录下仍然保留了一批同名 Markdown 文件。
  - 这些文件主要是编辑器占用或历史兼容遗留。
  - 规范上应以 `docs/` 下面的版本为准。
- 如果后续你关闭 VS Code 中对应标签页并解除文件占用，可以再把根目录的重复 Markdown 文件物理删除。
